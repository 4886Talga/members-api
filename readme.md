# Members API

> Beckend API for Members application

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings to your own

## Install Depenencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

#Run in prod mode
npm start
```
